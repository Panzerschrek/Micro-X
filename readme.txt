"Micro-X"  - 32k игра

-------------------------ИГРОВОЙ ПРОЦЕСС----------------------
 В процессе игры нужно летать по лабиринту, отбиваться от
монстров, собирать патроны, аптечки. Также, в лабиринте есть
особые объекты - икосаэдры. Чтобы победить, нужно собрать их
всех. Чтобы не заблудиться в лабиринте, можно посмотреть на
карту. Для этого надо найти её изображение на стене (в виде
икосаэдра из белых линий на чёрном фоне), подлететь к нему и
нажать TAB.


---------------------СИСТЕМНЫЕ ТРЕБОВАНИЯ---------------------

* Компьютер с операционной системой Windows XP или выше.
* Наличие Microsoft Visual C++ 2008 Redistributable в системе.
  Можно взять здесь:
  https://www.microsoft.com/ru-ru/download/details.aspx?id=29
* Наличие Microsoft DirectX 9.0c в системе. Можно взять здесь:
  https://www.microsoft.com/ru-ru/download/details.aspx?id=34429
* Поддержка OpenGL 3.3 в операционной системе/драйверах.
* Видеокарта с 512 Mb оперативной памяти.
* 1024 Gb оперативной памяти.
* Процессор архитектуры x86 с поддержкой SSE2.
* Экран с разрешением не меньше чем 800x600.


--------------------------УПРАВЛЕНИЕ--------------------------

W - ускоряться вперёд
S - ускоряться назад
A - ускоряться влево
D - ускоряться вправо
SPACE - ускоряться вверх
C - ускоряться вниз
M - включить/выключить захват мыши
Q - поворачиваться по часовой стрелке
E - поворачиваться против часовой стрелки
Левая клавиша мыши - стрелять
Колесо мыши - менять оружие
TAB - воспользоваться картой уровня (карты развешаны на стенах)
ESC - выход


-------------------ПАРАМЕТРЫ КОМАНДНОЙ СТРОКИ-----------------

--fullscreen запустить игру в полноэкранном режиме в текущем
 разрешении экрана.
--no-vsync не использовать вертикальную синхронизацию.
--invert-mouse-y инфертировать ось Y мыши при управлении обзором.
--sx [чувствительность] задать чувствительность мыши по оси X.
--sy [чувствительность] задать чувствительность мыши по оси Y.


--------------------------ИГРОВОЙ ЭКРАН-----------------------

* Зелёная полоска в правом нижнем углу - индикатор здоровья.
* Серые квадраты рядом с полосокй здоровья -
 дополнительные жизни.
* Цветной большой квадрат в левом нижнем углу - текущее оружие.
* Квадраты разных цветов и размеров в левом нижнем углу -
 доступные боеприпасы.
* Шестиугольники в верхнем левом углу - информация об
 икосаэдрах на уровне. Серый цвет - икосаэдр ещё не подобран,
 жёлтый - подобран.
